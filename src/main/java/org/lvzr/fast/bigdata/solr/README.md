SolrJ是操作Solr的Java客户端，它提供了增加、修改、删除、查询Solr索引的JAVA接口。SolrJ针对 Solr提供了Rest 的HTTP接口进行了封装， 
SolrJ底层是通过使用httpClient中的方法来完成Solr的操作。
